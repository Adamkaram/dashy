# نظام الألوان الموحد - 6 ألوان فقط

## الألوان الأساسية

### 1. **#ECE8DB** - Cream Light (بيج فاتح جداً)
- **الاستخدام**: خلفيات فاتحة، هيدر الصفحات
- **CSS Variable**: `--color-cream-light`
- **أمثلة**: `.page-header`, خلفيات الأقسام الفرعية

### 2. **#F0EBE5** - Beige (بيج فاتح)
- **الاستخدام**: خلفية الصفحة الرئيسية، خلفيات الأقسام
- **CSS Variable**: `--color-beige` أو `--background`
- **أمثلة**: `body`, `main`, خلفيات النماذج

### 3. **#8F6B43** - Brown Gold (بني ذهبي)
- **الاستخدام**: العناصر التفاعلية، الأزرار، الروابط عند التمرير
- **CSS Variable**: `--color-brown-gold` أو `--secondary`
- **أمثلة**: الأزرار الأساسية، الروابط النشطة، الأيقونات

### 4. **#53131C** - Burgundy (أحمر داكن/بني)
- **الاستخدام**: العناوين الرئيسية، التأكيدات، العناصر المهمة
- **CSS Variable**: `--color-burgundy` أو `--primary`
- **أمثلة**: `h1`, `h2`, عناوين الأقسام

### 5. **#46423D** - Gray Dark (رمادي داكن)
- **الاستخدام**: النصوص الأساسية، المحتوى الرئيسي
- **CSS Variable**: `--color-gray-dark` أو `--text-primary` أو `--foreground`
- **أمثلة**: `p`, `span`, النصوص العادية

### 6. **#363533** - Near Black (أسود مائل للرمادي)
- **الاستخدام**: النصوص الثانوية، التفاصيل
- **CSS Variable**: `--color-black` أو `--text-secondary`
- **أمثلة**: النصوص الصغيرة، التعليقات التوضيحية

## كيفية الاستخدام

### في CSS:
```css
.my-element {
  background-color: var(--color-beige);
  color: var(--text-primary);
  border-color: var(--secondary);
}
```

### في Tailwind/Inline Styles:
```tsx
<div className="bg-[#F0EBE5] text-[#46423D]">
  <h1 className="text-[#53131C]">عنوان</h1>
  <button className="bg-[#8F6B43] hover:bg-[#53131C]">زر</button>
</div>
```

## قواعد الاستخدام

1. **لا تستخدم أي ألوان أخرى** غير الـ 6 ألوان المحددة
2. **استخدم المتغيرات** بدلاً من القيم المباشرة عندما يكون ذلك ممكناً
3. **الشفافية مسموحة**: يمكنك استخدام `rgba()` مع هذه الألوان
4. **الأبيض والأسود**: يمكن استخدام `#FFFFFF` (أبيض نقي) و`#000000` (أسود نقي) للحالات الخاصة فقط
